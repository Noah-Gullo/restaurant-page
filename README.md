# Restaurant Page
## Link:
https://noah-gullo.github.io/restaurant-page/

## Description:
A mockup fantasy restaurant page containing three seperate sections: the homepage advertising the restaurant, the menu with images and descriptions of each dish, and contact page with fake hours.

## Skills:
This project was useful for me to test out bundlers, specifically webpack to create more effecient web applications that utilize a large number of module files without manually managing dependencies.

## Reflection:
This project was a good introduction to using Webpack as a bundler and definitely gave me insight into their benefits. Additionally, it was a good use of my JavaScript and HTML/CSS skills along with web design.
